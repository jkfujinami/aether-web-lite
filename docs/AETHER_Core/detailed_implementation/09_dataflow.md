## Part 9: データフロー図

### 9.1 メッセージ送信フロー

```
Alice                           DHT                    Mailbox Nodes (5個)                Bob
  |                              |                            |                            |
  |-- (1) FIND_VALUE(Bob_ID) --->|                            |                            |
  |<---- Bob's Prekey Bundle ----|                            |                            |
  |                              |                            |                            |
  |-- (2) X3DH Handshake --------|----------------------------|--------------------------->|
  |                              |                            |                     [Prekey消費]
  |                              |                            |                            |
  |-- (3) Double Ratchet暗号化 --|                            |                            |
  |      [plaintext → ciphertext]|                            |                            |
  |                              |                            |                            |
  |-- (4) Reed-Solomon分割 ------|                            |                            |
  |      [ciphertext → 5 shards] |                            |                            |
  |                              |                            |                            |
  |-- (5) Onion Routing ---------|                            |                            |
  |      [各shard を 3-Hop経由]   |                            |                            |
  |                              |                            |                            |
  |------------------------------------------PUT(shard_0) --->| M0                         |
  |------------------------------------------PUT(shard_1) --->| M1                         |
  |------------------------------------------PUT(shard_2) --->| M2                         |
  |------------------------------------------PUT(shard_3) --->| M3                         |
  |------------------------------------------PUT(shard_4) --->| M4                         |
  |                              |                            |                            |
  |                              |                            | [保存完了]                  |
```

### 9.2 メッセージ受信フロー

```
Bob                             Mailbox Nodes
  |                                  |
  |-- (1) LIST(Bob_ID) ------------->| M0, M1, M2, M3, M4
  |<---- [msg_id_1, msg_id_2, ...] --|
  |                                  |
  |-- (2) GET(msg_id_1) ------------>| M0
  |<---- shard_0 --------------------|
  |-- (2) GET(msg_id_1) ------------>| M1
  |<---- shard_1 --------------------|
  |-- (2) GET(msg_id_1) ------------>| M2
  |<---- shard_2 --------------------|
  |      [3シャード揃った！打ち切り]  |
  |                                  |
  |-- (3) Reed-Solomon復元 --------->|
  |      [3 shards → ciphertext]     |
  |                                  |
  |-- (4) Double Ratchet復号 ------->|
  |      [ciphertext → plaintext]    |
  |                                  |
  |-- (5) DELETE(msg_id_1, sig) ---->| M0, M1, M2, M3, M4
  |      [Burn-on-Read完了]          | [物理削除]
```
